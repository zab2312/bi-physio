# Satoshi Font Files

Ovdje stavite font datoteke za Satoshi font.

## Potrebne datoteke:

- `Satoshi-Regular.woff2` (ili `.woff` ili `.ttf`)
- `Satoshi-Medium.woff2` (ili `.woff` ili `.ttf`)
- `Satoshi-Bold.woff2` (ili `.woff` ili `.ttf`)

## Format:

Preferirani format je `.woff2`, ali podržani su i `.woff` i `.ttf` formati.

Nakon što dodate font datoteke u ovaj folder, font će se automatski učitati u aplikaciji.
